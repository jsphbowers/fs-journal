## Questions

1. What are ***lifecycle hooks***? What are lifecycle hooks used for?

Lifecycle hooks are simple computed states within vue that helps with the website maintaining a reactive state where if something changes then the website will live update.

2. How have you used lifecycle hooks in your afternoon projects?

Using lifecycle hooks will help with getting information to the page and making changes that will help the overall function of hte page.

3. What are ***mounting hooks***? When might you use them?

Mounting hooks can help with the initial load of the page or when changing pages within the site.
